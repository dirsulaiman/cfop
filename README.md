# CFOP
Rubik Cube algorithm collection with [CFOP](https://en.wikipedia.org/wiki/CFOP_method) method written in [LaTex](https://www.latex-project.org/help/documentation/).

#### How to Compile
`$ pdflatex main.tex`
>The name of output file will be **main.pdf**
